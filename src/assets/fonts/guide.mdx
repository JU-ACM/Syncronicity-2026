# Typography Guide

This project uses a mix of local branding fonts and Google Fonts. Below is the reference for applying them using Tailwind CSS.

## Available Font Families

| Font Family        | Tailwind Class     | Best Used For             |
| :----------------- | :----------------- | :------------------------ |
| **Bounded** | `font-bounded`     | Heavy Headings (Black 900)|
| **Unbounded** | `font-unbounded`   | Modern, wide display text |
| **Euclid Circular**| `font-euclid`      | Body text & UI elements   |
| **Montserrat** | `font-montserrat`  | General Purpose           |

---
