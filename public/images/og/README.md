# Open Graph Images

Place your social sharing images in this folder. Recommended sizes:

- **home.png** - 1200x630px - Home page OG image
- **event.png** - 1200x630px - Event pages OG image
- **problem.png** - 1200x630px - Problem statement pages OG image
- **default.png** - 1200x630px - Fallback OG image

## Image Guidelines

- Use 1200x630px for optimal display on all platforms
- Keep text minimal and readable at small sizes
- Include branding (logo, event name)
- Use high contrast colors
- Test with Facebook's Sharing Debugger: https://developers.facebook.com/tools/debug/
